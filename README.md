***Rustick***

## 介绍

Rustick 是一款开源的秒表计时器，使用 Rust 语言编写，基于 egui 库开发。

> 当前版本只适用于 Windows 系统。

### 日志记录

使用 PoloDB 库实现日志记录功能。
